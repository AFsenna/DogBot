APLIKASI DOGBOT

Target : Pemilik anjing baru (first-time dog owners) yang masih bingung mengenai dasar-dasar perawatan, jadwal grooming, dan nutrisi yang tepat
Kegunaan : Konsultasi personalisasi berbasis ras dan usia, solusi perilaku dengan positive reinforcement, dan sistem deteksi dini gejala darurat

install package npm
  npm install

run dengan : 
  node index.js
